1.小金鱼 - RoCEv2协议分析工具 简介
	小金鱼是一款轻量级的RoCEv2协议分析工具，专注于RDMA网络调试与验证。它能快速解析PCAP抓包文件，生成PSN时序图，帮助工程师定位乱序、丢包等网络问题。

	核心特性
	🚀 极速解析：7.5万包/87ms，约87万包/秒（零拷贝架构）

	📊 PSN时序图：自动生成多QP时序图，异常点自动标记

	🔍 智能分析：按五元组+QP聚合流，检测PSN连续性

	🎨 可视化：不同QP不同颜色，图例清晰对应

	📦 轻量：21M，解压即用，无需安装

3.系统要求
	Windows 10/11 (64位)

	无需安装VC运行库（已静态编译）
	
	Linux版本在规划中

4.使用方法
	下载并解压小金鱼_v1.0.zip

	运行 GoldenFish.exe

	点击“浏览”选择PCAP文件

	点击“PSN时序图”查看分析结果

	采样率：默认1000:1，可在设置中调整
    点击“时序详情”查看每条曲线对应的QP


6.示例文件
	软件包内置示例文件 rdma2-rxe-r6.18-tmp-10.pcap，可直接加载体验。
